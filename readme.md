# Template Docker Node Vue

# Installation guide

## What you need

| Runtime / Software |
| ------------------ |
| Docker             |
| Docker Compose     |

## Start the database with Docker

```sh
docker-compose up
```

## Start the express Api

```sh
cd api/
npm install
npm run dev
```

## Install the client

```sh
cd client/
npm install
```

## Start the client

```sh
npm start
```
